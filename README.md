# @codewell/reorder-array
Reorder an array

## Installation
```
npm install @codewell/reorder-array
```

## Basic usage
```JavaScript
import reorderArray from '@codewell/reorder-array';

reorderArray(['a', 'b', 'c'], 0, 2); // => ['b', 'c', 'a']
reorderArray(['a', 'b', 'c'], 2, 0); // => ['c', 'a', 'b']
```
